# GRC Portfolio — Tri-State Community Bancorp (Fictional)

A simulated governance, risk, and compliance (GRC) program built around 
a fictional $2.1B regional bank holding company headquartered in Cincinnati, Ohio.

## Frameworks Applied

- **NIST Cybersecurity Framework (CSF) 2.0** — Cybersecurity risk assessment 
  with Tier ratings across all six core functions (Govern, Identify, Protect, 
  Detect, Respond, Recover) and a 15-entry risk register
- **GLBA Safeguards Rule** — Compliance gap assessment across all 9 required 
  elements with executive summary and remediation recommendations
- **SOX IT General Controls (ITGC)** — Control testing over access provisioning, 
  access deprovisioning, and change management; includes a formal audit finding 
  in Condition/Criteria/Cause/Effect/Recommendation format
